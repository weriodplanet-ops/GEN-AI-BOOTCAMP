# GEN AI Bootcamp - Day 1 Homework: Multimodal AI

## Project Overview
Building an AI system for an e-commerce platform (Amazon/Flipkart) that understands products from images.

---

## Problem Statement

🧠 **Challenge**: Build an AI system that can understand a product using an image.

The system should be able to:
- Analyze product images
- Extract meaningful information
- Generate relevant descriptions and categories

---

## Your Task

### 1. Inputs
- **Product Image** - The visual input for analysis

### 2. Outputs
- Product category
- Product description

### 3. Model Selection
**Question**: What kind of model can understand both image and text?

**Hint**: Think about multimodal models like:
- CLIP
- BLIP
- Other vision-language models

### 4. System Design (High-Level)
**Question**: How will your system work step-by-step?

*Think through the pipeline:*
- Image input processing
- Model inference
- Output generation
- Post-processing

### 5. Evaluation
**Question**: How will you know if your model is good?

**Consider**:
- Accuracy of predictions
- Quality of generated descriptions
- User satisfaction metrics

### 6. Business Value
**Question**: Why is this useful?

*Consider*:
- Operational efficiency
- Customer experience
- Cost reduction
- Scalability

---

## Submission Format

✅ **Requirements**:
- Format: Google Doc / Notion / PDF

- Include diagrams (even rough sketches encouraged)
- Write clear answers to all 6 sections above
